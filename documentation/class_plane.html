Detailed Description

Opisuje plaska powierzchnie w 3D reprezentujaca powierzchnie marsa, plasczyzna docelowo rysuje sie w gnuplocie w zakresach [0,range_x] x [0,range_y], stad klasa ma pola tylko range_x i range_y.
