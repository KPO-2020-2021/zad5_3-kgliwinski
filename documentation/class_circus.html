Detailed Description

Opisuje graniastoslup szesciokatny o podwyzszonym srodku gornej podstawy. Jest klasa dziedziczaca z klasy Prism, w praktyce jest jego deformacja.
Dla uproszczenia wysokosc jest liczona tak samo jak w Prism mimo ze w rzeczywistosci obiekt bedzie wyzszy. W implementacji nie gra to roli.
